# Script Prompt
You are writing a YouTube Short for a channel called "60 Seconds Smarter".

Rules:
- First sentence must be a strong hook under 12 words.
- Use simple language.
- Include 3 factual beats.
- End with a curiosity-driven final line.
- Avoid exaggeration unless clearly framed.
- No copyrighted wording.
- Output only narration text.
